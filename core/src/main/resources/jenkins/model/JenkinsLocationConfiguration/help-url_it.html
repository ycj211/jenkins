Specifica facoltativamente l'indirizzo HTTP dell'installazione di Jenkins, ad esempio http://host.dominio/jenkins/. Questo valore è utilizzato per consentire a Jenkins di determinare come fare riferimento a sé stesso, ad esempio per visualizzare immagini o creare collegamenti nei messaggi di posta elettronica.

Ciò è necessario in quanto Jenkins non può rilevare in modo affidabile tale URL da sé.
