Per impostazione predefinita, Jenkins consente agli utenti di creare account autonomamente tramite il collegamento "Registrati" in alto a destra nella pagina. Se, tuttavia, si desidera impedire a utenti casuali di creare account e di controllare in modo rigido la creazione degli account utente, deselezionare questa casella di controllo.

Quando questa casella di controllo è deselezionata, la creazione degli account utente sarà compito di un utente con il ruolo di amministratore.

Per impostazione predefinita, Jenkins non utilizza una verifica CAPTCHA se un utente crea un account autonomamente. Se si desidera abilitare la verifica CAPTCHA, installare un plugin di supporto CAPTCHA, ad esempio il plugin JCaptcha di Jenkins.
