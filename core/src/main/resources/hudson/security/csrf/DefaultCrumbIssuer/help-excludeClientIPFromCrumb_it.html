Alcuni proxy HTTP filtrano le informazioni che il componente predefinito di emissione dei crumb utilizza per calcolare il valore del nonce. Se un proxy HTTP è situato fra il proprio browser e il proprio server Jenkins e si riceve una risposta 403 dopo l'invio di un modulo a Jenkins, selezionare quest'opzione potrebbe essere d'aiuto. L'utilizzo di quest'opzione rende il valore del nonce più facilmente falsificabile.
