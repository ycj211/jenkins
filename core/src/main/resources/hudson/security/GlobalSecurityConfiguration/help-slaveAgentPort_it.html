Jenkins utilizza una porta TCP per comunicare con gli agenti lanciati tramite JNLP. Normalmente questa porta è scelta casualmente per evitare collisioni, ma ciò renderebbe difficile mettere in sicurezza il sistema. Se non si utilizzano agenti JNLP, è raccomandato disabilitare questa porta TCP. In alternativa, è possibile specificare un numero di porta fisso per poter configurare il proprio firewall in modo appropriato.
