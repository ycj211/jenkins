L'archiviatore degli artefatti utilizza il org.apache.tools.ant.DirectoryScanner di Ant che, per impostazione predefinita, considera le differenze tra maiuscole e minuscole. Ad esempio, se il processo produce file *.hpi, il pattern "**/*.HPI" non riuscirà ad individuarli.

Quest'opzione può essere utilizzata per disattivare le differenze maiuscole/minuscole. Quando non è selezionata, il pattern "**/*.HPI" individuerà tutti i file *.hpi, o il pattern "**/cAsEsEnSiTiVe.jar" individuerà un file denominato caseSensitive.jar.
