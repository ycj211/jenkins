Archivia gli artefatti della compilazione (ad esempio, file ZIP da distribuire o file JAR) in modo da poterli scaricare in un secondo momento. I file archiviati saranno accessibili dalla pagina Web di Jenkins.
Normalmente, Jenkins mantiene gli artifatti di una compilazione fintantoché viene mantenuto il registro di compilazione stesso, ma se i vecchi artefatti non sono richiesti e si preferirebbe risparmiare spazio su disco, è possibile farlo.

Si noti che il tipo di lavoro Maven archivia automaticamente gli artefatti Maven prodotti. Gli artefatti configurati qui saranno archiviati insieme ad essi. L'archiviazione automatica degli artefatti può essere disabilitata dalle opzioni avanzate di Maven.
