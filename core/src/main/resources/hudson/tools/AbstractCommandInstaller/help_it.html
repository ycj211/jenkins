Esegue un comando della shell a scelta per installare lo strumento. Ad esempio, con Ubuntu, assumento che l'utente Jenkins sia elencato in /etc/sudoers: 

sudo apt-get --yes install openjdk-6-jdk

(In questo caso specificare ad es. /usr/lib/jvm/java-6-openjdk-i386 come directory home.)
