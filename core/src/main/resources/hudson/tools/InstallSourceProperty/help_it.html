Selezionare quest'opzione per consentire a Jenkins di installare questo strumento a richiesta.

Se si seleziona quest'opzione, verrà richiesto di configurare una serie di "programmi di installazione" per questo strumento, dove ogni programma di installazione definisce il modo in cui Jenkins proverà a installare questo strumento.

Per uno strumento indipendente dalla piattaforma (come Ant), configurare programmi di installazione multipli per un singolo strumento non ha molto senso, ma per uno strumento dipendente dalla piattaforma configurazioni multiple dei programmi di installazione consentono di eseguire uno script di installazione differente a seconda dell'ambiente dell'agente.
