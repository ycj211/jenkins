Se impostato, il nome visualizzato (facoltativo) verrà visualizzato nel progetto nell'interfaccia grafica Web di Jenkins. Dal momento che esso è utilizzato solo per scopi di visualizzazione, non è necessario che questo sia unico per ogni progetto.
Se il nome visualizzato non è impostato, l'interfaccia grafica Web di Jenkins visualizzerà il nome del progetto per impostazione predefinita.
