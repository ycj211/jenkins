Ignora le modifiche notificate dagli hook post-commit del sistema di controllo del codice sorgente.

Ciò può essere utile se si desidera impedire che alcuni processi di lunga durata (ad es. report) siano avviati a causa di ogni singolo commit, ma si desideri comunque eseguirli periodicamente se si sono verificate modifiche nel sistema di controllo del codice sorgente.

Si noti che quest'opzione richiede il supporto anche da parte del plugin del sistema di controllo del codice sorgente! Il plugin Subversion la supporta a partire dalla versione 1.44.
