Questo componente controlla lo spazio su disco disponibile per la directory temporanea e, se questo scende sotto una certa soglia, pone il nodo non in linea.

Gli strumenti e i test/le compilazioni Java spesso creano dei file nella directory temporanea e potrebbero non funzionare correttamente se non c'è spazio disponibile.

Più in dettaglio, questo componente controlla lo spazio su disco disponibile di una partizione che contiene la directory designata dalla proprietà di sistema java.io.tmpdir. Per verificare la sua posizione su un dato agente, aprire ${rootURL}/computer/SLAVENAME/systemInfo.
