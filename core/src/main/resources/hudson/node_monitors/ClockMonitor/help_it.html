Questo monitor controlla la differenza fra l'orario del master e quello dei nodi. Mentre Jenkins stesso, in generale, è in grado di tollerare le differenze di orario fra sistemi, le attività legate al controllo del codice sorgente e l'accesso ai file distribuito (come NFS o le condivisioni file Windows) tendono ad avere strani problemi quando i sistemi coinvolti hanno orari differenti.

Per mantenere gli orologi sincronizzati, fare riferimento a NTP.
